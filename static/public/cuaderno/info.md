# Info cuaderno

> Esta app decido incluirla como una app de notas para los trabajadores, pero principalmente para que los jefes puedan generar anuncios fáciles para sus usuarios.

## Requerimientos funcionales
### Genere, abra, guarde, edite y elimine archigos .txt
### Genere, abra, guarde, edite y elimine archigos .md
### Insertar imágenes
### Todo list

## Requerimientos no funcionales
### Guardado en su propia base de datos
- Nombre
- Contenido
- Autor
- Fecha
- Imagenes embebidas
